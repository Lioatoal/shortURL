# shortURL
a shorten URL tool, created by node.js + sqlite + redis

## Required
Nodejs >= 8.0.0 (Using async/await)

### command
```bash
# eslint
npm run lint

# Sync Table Schema
npm run sync-db

# test
npm run test

# development
npm run dev

# build for production
npm run build

# run as production (pm2)
npm start
```
